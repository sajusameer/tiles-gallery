# Project Name: TilesGallery

##  Project Purpose

TilesGallery is a modern web application where users can explore different aesthetic tiles, view details, and manage their profile. It provides authentication, search functionality, and a clean UI for browsing tile collections.

---

##  Live URL

👉 https://tiles-gallery-gamma.vercel.app

---

## ✨ Key Features

*  User Authentication (Login & Register)
*  Google Social Login (Better Auth)
*  Home page with Hero Slider & Featured Tiles
*  All Tiles page with search functionality
*  Tile Details (Private Route)
*  My Profile (Private Route)
*  Update Profile (Name & Image)
*  Protected Routes (Login required)
*  Toast Notifications (React Toastify)
*  Swiper Slider (SwiperJS - React)
*  Fully Responsive Design (Mobile + Desktop)

---

##  Route Summary

###  Public Routes

* `/` → Home
* `/all-tiles` → Tiles Gallery
* `/login` → Login Page
* `/register` → Register Page

###  Private Routes

* `/tile/[id]` → Tile Details
* `/my-profile` → User Profile
